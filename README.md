# Projeto Costs

## Para rodar o projeto use:

### `npm install`

Instala as dependências do projeto.

### `npm start`

Abra [http://localhost:3000](http://localhost:3000) para ver no navegador.

### `npm run backend`

Inicia o 'Backend'.
